# PS4 505-1252 WEBKIT Crash Test

A simple web page designed to test the PS4 505-1252 WebKit crash vulnerability
  

## Usage

1. Open this page in your browser on a PS4 firmware 505-1252
2. Click the **Start** button to enter fullscreen mode
3. Follow the instructions to press OPTIONS and refresh the page to test the WebKit crash
